# Neovim

My Own Neovim Configuration

### Installation:

```sh
    git clone https://github.com/NvChad/NvChad ~/.config/nvim --depth 1 && nvim
```

 Or:

```sh
    git clone git@github.com:ProgmRuanSilva/nvim.git ~/.config/nvim && nvim
```
